# chill
## Program
* [chill.c](chill.c): C program to calculate wind chill based on temperature and wind speed

Note: [chill.c](chill.c) must be compilied with the `-lm` flag:
```bash
mygcc chill.c -lm -o chill
```

## Testing
* [testing.sh](testing.sh): bash script to execute a sequence of commands using the chill program
* [testing.out](testing.out): output from testing script
